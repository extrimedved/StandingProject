## Описание проекта

Проект представляет собой REST API для работы с результатами матчей и получения турнирной таблицы чемпионата на выбранную дату.

В данном проекте были реализованы такие функции как: 
1. Регистрация результата сыгранного матча
2. Получение турнирной таблицы по сезону и дате
3. Реализация unit-тестов для проверки функций проекта

## Подготовительные действия

Для запуска проекта необходимо:

1. Установить JDK 21
2. Установить среду разработки (vs code, idea и т.д.)

## Информация о доступах

Приложение работает на порту 8080.

## Как запустить проект

1. После открытия проекта в среде разработки запускаем программу
2. Для удобства использования контроллеров переходим в swagger по ссылке:http://localhost:8080/swagger-ui/index.html.
